Julián Abaroa

C++ Developer and Reverse Engineer.

I am a C++ developer focused on systems programming, with experience building mods for “Halo: The Master Chief Collection”, specifically Halo Reach. These mods are primarily based on DLL proxy inyection, function hooking via MinHook, and reverse engineering using Cheat Engine, x64dbg and Ghidra. I’ve also worked on gametyping with ReachVariantTool, a tool that uses a very similar language to Megalo, which allows to create custom gametypes for Halo Reach that are compatible with consoles. Previously, I worked with C# and Unity while studying video game design at Instituto Profesional Arcos.

Artemis-MCC aims to train an AI agent for combat in Halo Reach (MCC). However, the bulk of the project is the extraction, reinterpretation, and delivery of real-time game data: from reading object and player tables, through semantic reconstruction of that raw data, to producing a custom tick containing everything relevant for AI training.

AutoTheater-MCC
Author Oct 2025 - Apr 2026

AutoTheater-MCC automates camera direction in Halo Reach (MCC)’s Theater mode, aiming to always show the most interesting action happening at any given moment during a match replay. It also includes a replay file manager for saving and loading replays across all Halo: The Master Chief Collection titles, plus a built-in recording system using FFmpeg.
